# 白塔行商记

一个 React + Vite 制作的奇幻商旅文字小游戏。玩家经营一支货车队，在大陆地点之间低买高卖、探索秘闻、处理随机事件、完成委托、经营势力关系，并与各地人物发展互动。

## 主要内容

- 多地点市场：不同城市、要塞、森林、遗迹和工坊会影响商品价格。
- 随机旅途事件：战斗、行情波动、货物损失、奖励、势力委托和人物邂逅会持续改变局势。
- 人物关系：可交谈、相处、送礼、合作、邀请同行、共同任务、表白和结婚。
- 装备与商会成长：装备提供战斗、贸易、探索、社交和容量加成；商会路线会形成长期被动。
- 地点秘闻：探索和旅途中可能发现地标，记录到本地地图册并获得一次性奖励。
- 存档导入导出：可复制文本保存和恢复当前局。

## 开发脚本

```bash
npm install
npm run dev
npm run lint
npm run build
```

构建产物输出到 `dist/`，该目录不提交到仓库。
